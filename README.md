# Bandaru Bhavana Sri — Portfolio
A clean, responsive personal portfolio website built with plain HTML, CSS, and
JavaScript (no frameworks). It showcases my background as an AI/ML
undergraduate, technical skills, projects, experience, education, and contact
details.

## Live Demo
https://bhavana-portfolio-uwge.onrender.com

## Features
- Responsive layout that works across desktop, tablet, and mobile
- Sticky navigation bar with smooth scrolling to each section
- Hero section with a quick intro and call-to-action buttons
- About section with a short bio and key facts (location, email, phone, languages)
- Skills section grouped into Programming, Libraries & Tools, and AI/ML & Other
- Projects section with tech-stack tags, key features, and links to live demos / code
- Experience & Leadership section covering internship work and certifications
- Education timeline
- Contact section with clickable email, phone, LinkedIn, and GitHub cards
- Mobile-friendly collapsible navigation menu

## Tech Stack
- HTML5
- CSS3 (custom properties, Flexbox, Grid, no frameworks)
- Vanilla JavaScript
- Google Fonts (Poppins, Inter)

### Run Locally
1. Clone or download this repository.
   ```bash
   git clone https://github.com/bhavana-0003/<your-portfolio-repo>.git
   cd portfolio2
   ```
2. Open the folder in VS Code.
3. Right-click `index.html` → **Open with Live Server**.
   (Or simply double-click `index.html` to open it directly in a browser.)

### Deploy
- **GitHub Pages**: Push this folder to a GitHub repo, then enable Pages in
  **Settings → Pages** (source: main branch).
- **Netlify**: Drag and drop the `portfolio2` folder onto
  [netlify.com/drop](https://app.netlify.com/drop) for an instant live link.
- **Render**: https://bhavana-portfolio-uwge.onrender.com

## Customization
- **Colors & fonts**: edit the `:root` variables at the top of `style.css`.
- **Content**: update text directly in `index.html`.
- **Add a project**: duplicate a `.project-card` block in the Projects section
  and update the title, description, tags, and links.
